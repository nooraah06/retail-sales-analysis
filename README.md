# Retail Sales Analysis & Dashboard 

### Project Objective
To analyze retail sales data and generate actionable business insights
using SQL and Python, with interactive visualization in Power BI.

###  Tools & Technologies
- SQL
- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI
- Matplotlib / Seaborn

### Project Workflow
1. Data loading and cleaning
2. SQL-based aggregation
3. Exploratory Data Analysis (EDA)
4. Sales trend and product performance analysis
5. Exported processed data to Power BI
6. Built interactive dashboards

###  Dashboard Features
- Monthly Sales Trends
- Category-wise Performance
- Revenue KPI Cards
- Interactive Filters

###  Key Insights
- Identified top revenue-generating categories.
- Observed seasonal sales patterns.
- Highlighted high-performing products.
- Provided data-driven recommendations.

###  Dataset
Retail sales dataset (CSV format).

###  Conclusion
This project demonstrates an end-to-end data analysis workflow
combining SQL querying, Python processing, and Power BI visualization.
